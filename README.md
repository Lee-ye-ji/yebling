### 💻 ADMIN
```
•  상품등록
• 상품수정
• 상품삭제
```
###  🛍️ SHOP
```
• 상품페이지
- 상품 검색
- 해당 품목 및 가격 조절 선택
• 상품디테일페이지
-  재고 부족 시 '품절 표시'
```
### 🛒 CART
```
• 장바구니
- 체크박스를 이용하여 여러개의 상품 동시 주문 가능 
- 삭제버튼을 클릭하여 장바구니에서 상품 제거 
- 가격과 개수에 따른 가격 표시 
- 장바구니 내의 total 값 표시 
```
